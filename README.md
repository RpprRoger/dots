dots plugin
=========
# options
```js
$(element).dots({
    char: '.',
    count: 3,
    speed: 500,
    start: true
});
```
# start
```js
$(element).dots(true);
```
# stop
```js
$(element).dots(false);
```
# Update options
```js
$(element).dots({
    char: '.',
    count: 3,
    speed: 500
});
```
